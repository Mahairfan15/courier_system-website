# 🚚 Courier System Website  

> 📦 A simple yet efficient courier management system built with **HTML, CSS, PHP, and MySQL** — designed to streamline parcel tracking and delivery management.  

---

## 🧭 Overview  

The **Courier System Website** helps manage and track parcels efficiently.  
It provides both user and admin perspectives to simulate a real-world courier company’s workflow.  

### 🔍 Users can:  
- Book and track courier parcels  
- View shipment details and status updates  
- Access a clean and responsive UI  

### 🧑‍💼 Admin can:  
- Manage courier records (Add / Update / Delete)  
- View all booking details in one dashboard  
- Monitor delivery status and performance
